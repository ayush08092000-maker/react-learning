# React Components Practice 🚀

This project is created using React + Vite to learn React basics.

This is a simple React project where I practiced:

* Functional Components
* JSX
* Component-based structure

- Functional Components (Most Important ✅ - modern React)
- Class Components (Old Method ❌ - Rarely used & More complex)

* Navbar Component
* Card Component

## 🛠️ Tech Used

* JavaScript
* HTML, CSS

## ▶️ How to Run

```
npm install

```
