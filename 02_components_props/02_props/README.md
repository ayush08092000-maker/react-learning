# 02_props (React)

This project demonstrates how **Props** and **Props Destructuring** work in React.

## 📌 Concepts Covered

* Passing data from parent to child (Props)
* Using props inside components
* Props Destructuring

## 📂 Components

* App.jsx → Parent component
* Cards.jsx → Displays data using props
* Props_Destructuring.jsx → Uses destructured props

## 🚀 Run Project

npm install
npm run dev

## 💡 Purpose

To understand how data flows between components in React.
