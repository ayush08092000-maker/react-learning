# 💼 Job Cards UI

A simple React project that displays job cards using reusable components and props.

## 🚀 Features

🔹 Reusable Card Component
🔹 Props-based dynamic data rendering
🔹 Clean and modern UI design
🔹 Multiple job listings (FAANG & Tech companies)
🔹 Responsive layout (optional if added)

## 🛠️ Tech Stack

* React JS
* JavaScript
* CSS

📸 Preview

This project shows job cards with:

* Company logo
* Job role
* Tags (Full-time / Part-time, Level)
* Salary
* Location

## ▶️ Run Project

```bash
npm install
npm run dev
```

## 🎯 Purpose

Practice of React components and props.
